# Engenharia do Lucro
## Pipeline de Dados ETL para Marketing Inteligente em Lavanderia Self-Service
##### Trabalho de conclusão da sprint de Engenharia de Dados, disciplina do curso de pós-gradução em Ciências de Dados e Analytic (PUC-Rio)

Este MVP tem como objetivo construir o pipeline de dados para analisar o comportamento de consumo dos usuários de uma lavanderia self-service, onde o cliente realiza todos os serviços de forma autônoma, desde o cadastro, pagamento, uso das máquinas, até a dobradura das suas próprias roupas.

Antes de prover qualquer insight ou análise, é preciso que o dado passe por diversas etapas. Nosso intuito é, portanto, extrair, transformar e carregar os dados referentes algumas das análises que fazem parte da nossa rotina para aumentar a lucratividade através de campanhas de marketing mais eficazes!

Portanto, iremos preparar os dados de forma que eles respondam às seguintes perguntas de negócio:

1. qual o ticket médio do clientes, ou seja, qual a média de consumo mensal em reais;
2. métrica de ROAS (retorno sobre anúncios pagos) para entender se estamos tendo lucro ou prejuízo com esse investimento.
3. saber quem são os top 20 clientes que mais consomem para que possamos engajá-los com programas de fidelidade e até premiações;
4. total de novos usuários adquiridos por mês para acompanhar a eficácia das campanhas e enxergar se há outliers e o porquê dos mesmos.
